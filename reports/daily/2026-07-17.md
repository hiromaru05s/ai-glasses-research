## 技術・開発者向け情報（新規のみ）

### SDK・API更新

- **[🟡] Android XR `SceneCore` が初の beta に到達** — 3D scene graph を扱う Jetpack ライブラリ `androidx.xr.scenecore` が `1.0.0-beta01` として公開された。
  - **事実**: 2026年7月15日付の `XR SceneCore | Jetpack | Android Developers` で `1.0.0-beta01` が公開され、`Session.create` の suspend 化、`AnchorEntity` から `AnchorSpace` への名称変更、`SurfaceEntity.Shape.Quad` の `cornerRadius` 公開などの API 変更が記載された。
  - **推測 / 含意**: Android XR のシーン制御 API は実験段階から安定化フェーズに入っており、社内サンプルや検証コードが `Session.create` の呼び出し方やアンカー関連型名に依存しているなら追従コストを今のうちに見積もるべき。
  - 📎 [XR SceneCore | Jetpack | Android Developers](https://developer.android.com/jetpack/androidx/releases/xr-scenecore) — Android Developers

- **[🟡] Android XR `Projected` が権限要求 API を刷新** — 接続型 XR デバイス向け `androidx.xr.projected` が `1.0.0-alpha10` になり、権限取得フローが更新された。
  - **事実**: 2026年7月15日付の `XR Projected | Jetpack | Android Developers` で `1.0.0-alpha10` が公開され、`ProjectedPermissionsResultContract` の非推奨化と `ProjectedActivityCompat.requestPermissions()` の追加が案内された。
  - **推測 / 含意**: companion / projected 型の Android XR 体験を試している場合、権限まわりのサンプル実装は近いうちに古くなる。許可ダイアログ導線を持つ PoC は API 差し替え前提で見直した方が安全。
  - 📎 [XR Projected | Jetpack | Android Developers](https://developer.android.com/jetpack/androidx/releases/xr-projected) — Android Developers
