## デバイス・メーカー動向（新規のみ）

### Meta

- **🟡 Ray-Ban Meta / Oakley Meta が 2026年5月21日から日本販売開始** — Meta と EssilorLuxottica が日本投入を正式発表。Ray-Ban Meta (Gen 2) に加え Oakley Meta も同時展開し、サングラスだけでなく度付き Optics も含めた販売体制を敷く。
  - 事実: Meta は日本発売日を 2026年5月21日と明記し、Wayfarer / Skyler / Headliner と Optics 系フレームを案内した。日本では一部機能が未提供または順次展開とも注記している。
  - 推測: 日本での店頭認知と処方レンズ対応が進むと、B2C での「AIグラス体験」の基準が Meta 起点で固定化しやすい。国内 PoC 提案でも Meta 比較が避けにくくなる。
  - 📎 [Ray-Ban Meta (Gen 2)およびOakley Meta 5月21日より日本でも販売開始](https://about.fb.com/ja/news/2026/05/ai-glass-launch/amp/) — Meta

- **🟡 Meta が AIグラス向けアクセシビリティ機能を拡張** — Be My Eyes 連携強化、通話中の音声操作、ワンタッチショートカット、表示付きモデルでの captioned calls を紹介。単なる撮影デバイスではなく支援技術としての訴求を強めた。
  - 事実: Meta は Ray-Ban Meta Optics / Oakley Meta Vanguard でアクションボタンのカスタム起動、Be My Eyes 通話起動、音声での通話操作、表示付きモデルでのリアルタイム字幕を案内した。
  - 推測: 今後の競争軸はカメラ/音声性能だけでなく、障害当事者向け支援機能や第三者アプリ接続まで含む「日常インフラ性」に広がる。
  - 📎 [Our AI Wearables Are “Changing the Game” for Disabled People](https://about.fb.com/news/2026/05/meta-ai-wearables-changing-the-game-for-disabled-people/) — Meta Newsroom

### Google / Android XR

- **🔴 Google / Samsung が Gemini 搭載グラスの製品像を提示** — 音声中心の `audio glasses` と、情報表示を伴う `display glasses` を区別して説明し、ナビ、通知要約、写真撮影、翻訳、アプリ連携をハンズフリー体験として前面に出した。
  - 事実: Google は `audio glasses` と `display glasses` の2カテゴリを明示し、Gemini による視覚理解、ルート案内、メッセージ要約、写真撮影、翻訳、Uber などのアプリ連携を列挙した。Samsung も同日に Galaxy エコシステム連携を説明した。
  - 推測: Android XR は「フル没入型ヘッドセット」だけでなく「スマホ連携グラス」を主戦場に据え始めた可能性が高い。R&D の優先検討対象は 3D UI より companion UX に寄る。
  - 📎 [Intelligent eyewear is coming this fall](https://blog.google/products-and-platforms/platforms/android/android-xr-io-2026/) — Google Blog
  - 📎 [Samsung and Google Give First Look at New Intelligent Eyewear](https://news.samsung.com/uk/samsung-and-google-give-first-look-at-new-intelligent-eyewear) — Samsung Newsroom U.K.

### Apple

- **🟡 Apple が Vision Pro のアクセシビリティ機能を拡張** — 自動生成字幕を Apple Vision Pro に広げ、視線入力で対応車いすを操作できる新機能も予告。Vision Pro を医療・支援用途へ寄せる更新。
  - 事実: Apple は字幕自動生成が Apple Vision Pro でも使えると案内し、Vision Pro の視線追跡を使った対応電動車いすの操作機能を発表した。米国では Tolt / LUCI の代替ドライブシステムから開始予定。
  - 推測: Apple は短期では廉価グラスではなく、高精度センサーを活かすアクセシビリティ/業務用途で Vision Pro の意味づけを強める構えに見える。
  - 📎 [Apple unveils new accessibility features, and updates with Apple Intelligence](https://www.apple.com/newsroom/2026/05/apple-unveils-new-accessibility-features-and-updates-with-apple-intelligence/) — Apple Newsroom

## 技術・開発者向け情報（新規のみ）

### SDK・API更新

- **🔴 Android XR Developer Preview 4 公開** — `AI glasses` を `audio glasses`、`display AI glasses` を `display glasses` に名称整理し、Device Availability API、Project Aura 向け早期アクセス、Jetpack XR コアライブラリの Beta 移行予告を含む大きめの更新。
  - 事実: Google は DP4 で XR Runtime / Jetpack SceneCore / ARCore for Jetpack XR の Beta 移行予告、Jetpack Projected の Device Availability API、グラス向け API 強化を案内した。Unreal Engine / Godot の公式対応も同日ラインアップに入った。
  - 推測: Android XR の API 名称が整理されたことで、社内設計文書でも「AIグラス」という曖昧語より `audio/display glasses` 前提で要件分解した方が実装と揃いやすい。
  - 📎 [Updates to the Android XR SDK: Introducing Developer Preview 4](https://developer.android.com/blog/posts/updates-to-the-android-xr-sdk-introducing-developer-preview-4) — Android Developers Blog

- **🟡 Android XR Developer Catalyst Program の応募開始** — Google が Project Aura や `intelligent eyewear` 向けに、開発キット、技術支援、場合によっては grants を提供する開発者プログラムを開始した。
  - 事実: 対象は今後 6〜12カ月で Android XR 向けアプリ公開を目指す開発者で、wired XR glasses と intelligent eyewear の両方を含む。Kotlin / Unity / Unreal / Godot をサポート対象としている。
  - 推測: 2026年内の Android XR 市場は、ハード発売と同時に「初期アプリ不足」を避けるため、Google 主導で開発者獲得を急いでいる。
  - 📎 [Build for the future with the Android XR Developer Catalyst Program — Apply now!](https://android-developers.googleblog.com/2026/05/apply-android-xr-developer-catalyst.html) — Android Developers Blog

- **🟡 visionOS 26.6 beta 配信開始** — Apple Developer が `visionOS 26.6 beta (23O5728e)` を 2026年5月26日付で公開。新規リリースノート自体は大きくないが、WWDC26 前の安定化フェーズに入った。
  - 事実: Apple Developer は 2026年5月26日に `visionOS 26.6 beta` を配信開始し、リリースノートでは「この更新向けの新しいリリースノートはない」と案内している。
  - 推測: 直近の Vision Pro 開発者向け変化は新機能追加よりも、WWDC 前の微調整とテストサイクル維持が中心とみられる。
  - 📎 [visionOS 26.6 Beta Release Notes](https://developer.apple.com/documentation/visionos-release-notes/visionos-26_6-release-notes) — Apple Developer Documentation
