## Step 1: Google Drive MCPを接続（任意）

レポートを自動でDriveに格納したい場合:
1. Coworkの設定からMCPセクションを開く
2. 「Google Drive」を検索して「Connect」
3. Googleアカウントで認証

未接続でもレポートはローカルに保存される。

---

## Step 2: Google Driveにフォルダ作成（任意）

```
AIグラス市場レポート/
├── daily-delta/          ← 日次デルタレポート
└── biweekly-summary/     ← 隔週サマリー
```

## Step 3: GitHubリポジトリ作成

1. https://github.com/new → Private で作成
2. SSH鍵を設定（未設定の場合）:
```powershell
ssh-keygen -t ed25519 -C "maruru836818@gmail.com"
cat ~/.ssh/id_ed25519.pub
# → GitHubのSSH keys設定に登録
```

---

## Step 4: Git初期化

```powershell
cd C:\path\to\組織構築\org-glass-market
git init
git remote add origin git@github.com:YOUR_USERNAME/glass-market-research.git
git add .
git commit -m "init: AIグラス市場シンクタンク"
git push -u origin main
```

---

## Step 5: 初回実行

サイドバー「Scheduled」から:
- `glass-market-daily` → 「Run now」
- 権限承認ダイアログが出るので全て承認

一度承認すれば以降の自動実行では承認不要。
